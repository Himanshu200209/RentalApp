# 🏠 Rental Platform (Airbnb Clone)

A backend-heavy rental listing platform where users can create, view, edit, and delete rental listings. Built using **Node.js**, **Express**, and **MongoDB**, with server-side rendering using **EJS**.

## 🔗 GitHub Repo
[https://github.com/Himanshu200209/RentalApp](https://github.com/Himanshu200209/RentalApp)

## 🚀 Tech Stack
- Node.js
- Express.js
- MongoDB & Mongoose
- EJS
- Bootstrap
- REST APIs

## 🔧 Features
- Full CRUD functionality
- Add/Edit/Delete property listings
- Server-rendered pages using EJS
- Image support and responsive layout
- Modular route architecture

## 📦 Installation
```bash
npm install
node app.js
